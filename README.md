# 🍕 Pizza Sales Analysis Dashboard | Power BI

## 📌 Project Overview

This project presents an interactive **Pizza Sales Analysis Dashboard built using Microsoft Power BI**. The report provides business insights into pizza sales performance by analyzing revenue, order trends, customer preferences, product performance, and category-level sales distribution.

The dashboard helps identify sales patterns, understand customer buying behavior, and evaluate the performance of individual pizza products.

---

## 🎯 Business Objective

The objective of this project is to analyze pizza sales data and create an analytical dashboard that helps stakeholders:

* Monitor overall sales performance
* Understand daily and monthly order trends
* Identify top-performing and underperforming pizzas
* Analyze customer preferences by pizza category and size
* Support data-driven business decisions

---

## 📊 Key Performance Indicators (KPIs)

The dashboard tracks the following metrics:

| KPI                      | Description                                       |
| ------------------------ | ------------------------------------------------- |
| Total Revenue            | Total sales value generated from all pizza orders |
| Average Order Value      | Average revenue generated per order               |
| Total Pizzas Sold        | Total quantity of pizzas sold                     |
| Total Orders             | Total number of customer orders                   |
| Average Pizzas per Order | Average number of pizzas purchased per order      |

---

## 📈 Dashboard Features

### 1. Sales Overview Dashboard

Provides a high-level view of pizza business performance through KPI cards and sales insights.

Key metrics include:

* Revenue performance
* Order volume
* Pizza quantity sold
* Customer ordering behavior

---

### 2. Order Trend Analysis

Visual analysis of order patterns:

* **Daily Order Trend**

  * Shows fluctuations in order volume across days
  * Helps identify sales patterns

* **Monthly Order Trend**

  * Highlights changes in order activity over time
  * Helps understand seasonal sales movement

---

### 3. Sales Distribution Analysis

The dashboard analyzes sales contribution across different dimensions:

* Pizza Category Performance
* Pizza Size Preferences
* Category-wise quantity sold

These insights help understand customer preferences and product demand.

---

### 4. Best & Worst Sellers Analysis

The report identifies pizza products based on:

### Top 5 Sellers

Analyzed using:

* Revenue generated
* Quantity sold
* Number of orders

### Bottom 5 Sellers

Helps identify:

* Low-performing products
* Improvement opportunities
* Products requiring business attention
